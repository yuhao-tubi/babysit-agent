Default to a **light theme** for any generated HTML, UI, or diagram unless the user explicitly asks for dark.

**Why:** User stated this as a standing default after asking to convert a dark-themed deliverable to light.

**How to apply:** Choose light backgrounds and dark-on-light text by default; only go dark when the brief or user requests it.
